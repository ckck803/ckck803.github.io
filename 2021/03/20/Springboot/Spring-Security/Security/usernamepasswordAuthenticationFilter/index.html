UsernamePasswordAuthenticationFilter

AbstractAuthenticationProcessingFilterлҘј мғҒмҶҚн•ң к°қмІҙлЎң Usernameкіј PasswordлҘј мқҙмҡ©н•ҙ ліҙм•ҲмқёмҰқмқ„ м§„н–үн•  л•Ң мӮ¬мҡ©н•ҳлҠ” FilterмқҙлӢӨ.

1. Authentication к°қмІҙ мғқм„ұ

  • мӮ¬мҡ©мһҗ мҡ”мІӯмңјлЎң л¶Җн„° usernameкіј passwordлҘј к°Җм ёмҳЁлӢӨ.
  • usernameкіј passwordлҘј Authenticationмқ„ кө¬нҳ„н•ң UsernamePasswordAuthenticationToken к°қмІҙлҘј мғқм„ұ
  • мғқм„ұлҗң Authentication к°қмІҙлҠ” AuthenticationManager к°қмІҙлҘј нҶөн•ҙ мқёмҰқмқ„ м§„н–үн•ңлӢӨ.
  • Spring Securityм—җм„ңлҠ” AuthenticationManager к°қмІҙлҘј кө¬нҳ„н•ң ProviderManager к°қмІҙлҘј мӮ¬мҡ©н•ңлӢӨ.

ProviderManager hierarchy

к°Ғ ProviderManagerм—җлҠ” AuthenticationProvider к°қмІҙл“Өмқҙ мһҲлӢӨ. л§Ңм•Ҫ нҳ„мһ¬мқҳ ProviderManagerм—җ м Ғм Ҳн•ң AuthenticationProviderк°Җ м—Ҷмқ„ кІҪмҡ° Parent ProviderManagerлЎң мқҙлҸҷн•ҳкІҢ лҗңлӢӨ. Parent ProviderManager м—ӯмӢң AuthenticationProvider к°қмІҙл“Өмқҙ мЎҙмһ¬н•ңлӢӨ. м—¬кё°м„ңмқҳ ParentлҠ” мғҒмҶҚ к°ңл…җмқҙ м•„лӢҲлқј fieldк°’мңјлЎң parentлҘј к°Җм§Җкі  мһҲлӢӨ.

2. ProviderManagerлҘј нҶөн•ң мқёмҰқ м§„н–ү

ProviderManager к°қмІҙлҠ” мқёмҰқмқ„ м§Ғм ‘ м§„н–үн•ҳлҠ” н•ҳм§Җ м•ҠлҠ”лӢӨ. лҢҖмӢ , мқёмҰқ мң„н•ң лӢӨм–‘н•ң AuthenticationProvider к°қмІҙл“Өмқ„ к°Җм§Җкі  мһҲкі  AuthenticationProvider к°қмІҙм—җкІҢ мқёмҰқмқ„ мң„мһ„н•ҳлҠ” нҳ•нғңлЎң мқёмҰқмқ„ м§„н–үн•ңлӢӨ.

  • ProviderManagerм—җм„ң м Ғм Ҳн•ң AuthenticationProvider к°қмІҙлҘј к°Җм ёмҳЁлӢӨ.
  • к°Җм ёмҳЁ AuthenticationProviderлҘј мқҙмҡ©н•ҙ Authenticationк°қмІҙ мқёмҰқмқ„ м§„н–ү
  • мқёмҰқмқҙ мҷ„лЈҢ лҗҳл©ҙ мғҲлЎңмҡҙ Authentication к°қмІҙлҘј л°ҳнҷҳн•ҳкІҢ лҗңлӢӨ.
  • л§Ңмқј нҳ„мһ¬ ProviderManagerм—җм„ң м Ғм Ҳн•ң AuthenticationProvider к°қмІҙк°Җ м—ҶлӢӨл©ҙ л¶ҖлӘЁ ProviderManagerлЎң мқҙлҸҷн•ҙ мқёмҰқмқ„ мӢңлҸ„н•ңлӢӨ.

мқёмҰқ м „ Authentication к°қмІҙ

мқёмҰқ м „ Authenticatin к°қмІҙлӮҙмқҳ principalм—җлҠ” usernameмқҙ credentialsлҠ” passwordк°Җ л“Өм–ҙмһҲкі , мқёмҰқ мң л¬ҙлҘј лӮҳнғҖлӮҙлҠ” authenticated к°’мқҙ falseлЎң лӮҳнғҖлӮҳ мһҲлӢӨ.

мқёмҰқ нӣ„ Authentication к°қмІҙ

мқёмҰқмқ„ м§„н–үн•ң нӣ„ мқёмҰқ м „кіј лӢӨлҘҙкІҢ Authenticatin к°қмІҙмқҳ principalм—җлҠ” User к°қмІҙк°Җ л“Өм–ҙк°”кі  authenticated к°’мқҙ trueк°Җ лҗҗлӢӨ. мөңмў…м ҒмңјлЎң л°ҳнҷҳ лҗ  л•ҢлҠ” Authenticatin к°қмІҙм—җм„ң credentialsлҘј м§Җмҡ°кі  л°ҳнҷҳн•ңлӢӨ.

3. мқёмҰқ мҷ„лЈҢ нӣ„ мһ‘м—…

мқёмҰқмқҙ мҷ„лЈҢлҗң нӣ„ мһ‘м—…мңјлЎңлҠ” Sessionм—җ лҢҖн•ң мІҳлҰ¬мҷҖ SecurityContextHolderм—җ Authentication к°қмІҙлҘј л“ұлЎқн•ҳлҠ” мқјмқҙ лӮЁм•„ мһҲлӢӨ.
