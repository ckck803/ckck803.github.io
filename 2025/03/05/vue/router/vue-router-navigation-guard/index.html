[Vue 3] Router - 네비게이션 가드

2025-03-05
2025-04-16

이 예제는 다음과 같은 주요 네비게이션 가드를 보여줍니다:

전역 가드

  • beforeEach: 모든 라우팅 전에 실행 (인증 체크 예제 포함)
  • beforeResolve: 컴포넌트 렌더링 전 비동기 작업
  • afterEach: 라우팅 완료 후 작업

라우터별 가드

  • beforeEnter: 특정 라우트 진입 전 (About 페이지 예제)

컴포넌트 내 가드

  • beforeRouteEnter: 컴포넌트 진입 전
  • beforeRouteUpdate: 같은 컴포넌트 내 라우트 변경 시
  • beforeRouteLeave: 컴포넌트 떠나기 전

주요 특징

인증 상태 체크 예제
비동기 데이터 로딩 처리
페이지 이동 확인 다이얼로그
콘솔 로그로 가드 실행 순서 추적
컴포넌트별 데이터 로딩 로직
이 코드를 사용하려면:

Vue Router 설치: npm install vue-router@4
위 파일 구조를 따라 디렉토리 구성
실제 프로젝트에서는 인증 로직과 데이터 로딩 부분을 실제 API 호출로 대체
