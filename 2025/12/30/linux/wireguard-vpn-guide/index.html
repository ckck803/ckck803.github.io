WireGuard 개요

WireGuard는 최신 VPN 프로토콜로, 기존 IPsec 이나 OpenVPN 보다 훨씬 빠르고 간단하며 안전합니다. 리눅스 커널에 통합되어 있어 성능이 우수하고, 설정이 간단하며, 암호화 기술이 현대적입니다.

WireGuard의 장점

  • 간결함: 약 4,000줄의 코드 (OpenVPN은 100,000줄 이상)
  • 빠른 속도: 커널 레벨에서 작동하여 높은 성능
  • 최신 암호화: ChaCha20, Poly1305, Curve25519 사용
  • 간단한 설정: 공개키/비밀키 방식으로 설정이 단순함
  • 크로스 플랫폼: Linux, Windows, macOS, iOS, Android 지원

클라이언트 키 생성 (Windows)

Windows에서는 GUI가 자동으로 키를 생성하거나, PowerShell에서 직접 생성할 수 있습니다:

방법 1: GUI에서 자동 생성

  1. WireGuard 앱에서 Add empty tunnel… 클릭
  2. 자동으로 PrivateKey와 PublicKey가 생성됨
  3. 생성된 PublicKey를 복사하여 서버 설정에 추가

방법 2: PowerShell에서 수동 생성

중요: 생성된 공개키를 서버의 /etc/wireguard/wg0.conf 파일에 Peer 섹션으로 추가해야 합니다.

연결 설정

  1. Add Tunnel 버튼 클릭
  2. Add empty tunnel… 선택하거나 설정 파일 직접 import
  3. 다음 내용으로 설정 (자동 생성된 PrivateKey 사용 또는 위에서 생성한 키 입력):
  1. Save 클릭
